# Awesome Skills

A curated collection of skills for [Claude Code](https://claude.com/claude-code).

## What are Skills?

Skills are reusable instruction sets that extend Claude Code's capabilities. They provide domain-specific knowledge and workflows that Claude can invoke when needed.

## Available Skills

| Skill | Description | Version |
|-------|-------------|---------|
| [skill-maker](./skill-maker/) | Meta-skill for creating and validating Claude Code skills | 1.1.0 |
| [github-flow](./github-flow/) | Complete GitHub workflow: commits, versioning, changelogs, releases with auto-checks | 1.0.0 |

## Installation

Download the `.skill` file and place it in your Claude Code skills directory:

```bash
# Default location
~/.claude/skills/
```

## Contributing

To add your skill to this collection:

1. Fork this repository
2. Create a folder for your skill
3. Add your `.skill` file and checksum
4. Include a README with installation instructions
5. Submit a pull request
